# Product Requirements Documents

This directory contains PRDs for Open CIS features and capabilities.

## PRD Index

- [PRD-0001: Admin Interface](./0001-admin-interface.md) - openEHR/EHRBase management UI for templates, EHRs, compositions, and AQL queries
- [PRD-0002: Patient CRUD Operations](./0002-patient-crud.md) - Complete patient creation, update, and deletion functionality
- [PRD-0003: Encounter CRUD Operations](./0003-encounter-crud.md) - Complete encounter management for tracking patient visits and clinical interactions
- [PRD-0004: Vital Signs Chart with openEHR Transparency](./0004-vital-signs-chart.md) - Vital signs recording/charting with blood pressure and pulse, plus openEHR metadata inspection
